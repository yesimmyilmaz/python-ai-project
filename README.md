# Chess AI Project 

This is a Python-based chess game where a player can play against an AI.  
The project includes both a **terminal version** and a **GUI version** built with Pygame.

---

##  Features
- Play chess against computer AI
- Two modes:
  ✅ Terminal chess  
  ✅ GUI chess with interactive board
- AI powered by `python-chess` engine
- Move validation and legal move generation
- Simple and clean UI

---

## Requirements

Install dependencies before running:

```bash
pip install pygame python-chess
